# bittest
### 功能
- http请求，类postman功能，可以进行保存、收藏、历史记录
- 脚本在线编写，进行爬虫测试、或者变量组合加工的。支持jvm的所有脚本。目前主要使用groovy
- 延时功能。可以设置线程的执行sleep时间。
- 在线组装测试用例。将http接口、脚本等以上功能组合起来，进行快速回归测试。
- 在线组装测试任务。将用例组合起来，结合集群定时任务，可以进行分钟级别的定时跑批测试。
- 回调接收。单独提供回调通知接收接口，形成整个请求周期的闭合。例如支付收单接口、支付结果通知。
- 提供系统配置分门别类。将用例、任务归纳不同系统下。
- 变量。可以设置系统、任务、用例三个级别的请求变量设置，用于请求http或者执行脚本，另外的变量来源是从http的response返回里通过正则表达式提取出来、或者脚本运算加工的变量。当然http的执行，也有前置条件，前置条件较为复杂时，可以现在脚本功能里进行加工好。
